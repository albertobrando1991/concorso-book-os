# Implementation plan

## Milestone 1 - Foundation
- Scaffold app, env, TypeScript, tests and Git.
- Create vault structure and LLM Wiki operating schema.
- Add artifact and Knowledge memory files.

## Milestone 2 - Knowledge core
- Implement markdown frontmatter utilities.
- Implement heading append/replace operations.
- Implement local file store and Obsidian client.
- Keep raw sources immutable and route edits to wiki layers.

## Milestone 3 - Agents
- Ingest Agent classifies and stores sources.
- Knowledge Maintainer updates topic/entity pages and index.
- Book Writer compiles chapters from consolidated wiki layers.
- Quiz, Lint and Review agents produce learning and quality outputs.

## Milestone 4 - Product alignment
- Import `Metodo_BANDO_Progetto_Editoriale.md` as canonical source.
- Make `Il Metodo BANDO` the active book.
- Use BANDO, Bando Decoder, Moduli Profilo, Piano 30/60/90 and Diario Errori as structural anchors.

## Milestone 5 - Verification
- Run typecheck, unit tests and build.
- Capture UI screenshot if browser dependencies are available.
- Create GitHub repository after CLI authentication is restored.

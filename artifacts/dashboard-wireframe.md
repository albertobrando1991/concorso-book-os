# Dashboard wireframe

```text
+---------------------------------------------------------------+
| Sidebar                | Topbar: vault status + active book    |
| - Overview             |---------------------------------------|
| - Sources              | Metrics: sources, topics, chapters    |
| - Topics               | conflicts, stale, low confidence      |
| - Books                |---------------------------------------|
| - Agents               | Pipeline status | Agent queue         |
| - Quality              |---------------------------------------|
| - Logs                 | Sources table | Topics | Books       |
| - Settings             |---------------------------------------|
|                        | Quality issues | Logs + Artifacts     |
+---------------------------------------------------------------+
```

The dashboard is an operational control center, not a CRUD UI. It shows what entered the system, what was understood, what changed, which chapters were touched and what needs review.

# internet web posta elettronica

## Sintesi

Internet e web non coincidono: Internet e' l'infrastruttura di rete, il web e' un servizio basato su risorse accessibili tramite browser e URL. La posta elettronica usa protocolli specifici; la PEC aggiunge ricevute e valore legale nel sistema italiano.

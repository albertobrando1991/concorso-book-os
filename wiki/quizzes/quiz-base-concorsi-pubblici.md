# Quiz base concorsi pubblici

## Domande a risposta multipla
1. Da quali layer devono derivare i capitoli finali?
   - A. Raw sources soltanto.
   - B. Source summaries, topic pages ed entity pages consolidate.
   - C. Chat temporanee.
   - D. File non tracciati.
   - Risposta corretta: B.

2. Quale pagina e' append-only?
   - A. `log.md`.
   - B. Un capitolo.
   - C. Una raw source.
   - D. Una dashboard.
   - Risposta corretta: A.

## Flash recap
- Il libro e' una proiezione del wiki.
- Le source notes rendono tracciabili le fonti.
- Il lint segnala gap, orfani e bassa copertura.

## Errori tipici
- Usare raw sources come output finale.
- Non aggiornare topic pages dopo ingest.
- Non marcare capitoli impattati da nuove fonti.

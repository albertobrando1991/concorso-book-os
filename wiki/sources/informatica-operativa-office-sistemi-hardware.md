# Informatica operativa - office, sistemi operativi, file e hardware

## Sintesi

Questa fonte consolidata copre la parte piu ricorrente della banca dati concorsuale: videoscrittura, fogli elettronici, presentazioni, database d'ufficio, comandi base, file, cartelle, directory, estensioni, desktop, sistemi operativi, periferiche, CPU, RAM, ROM, memoria di massa, bit/byte, formati e codifica.

La base ufficiale e' costituita dai syllabus ICDL e dalle guide LibreOffice; per gestione file e sistemi operativi sono stati aggiunti riferimenti Microsoft Windows e Ubuntu. Per formati e codifica sono presenti IANA Media Types e Unicode.

## Nuclei ad alta frequenza

- Word/videoscrittura: creazione, modifica, salvataggio, formattazione, tabelle, immagini, stampa, esportazione.
- Excel/fogli elettronici: celle, riferimenti, formule, funzioni, grafici, ordinamento, filtri, fogli, cartelle di lavoro.
- Access/Base/database d'ufficio: tabelle, campi, record, relazioni, query, maschere, report.
- PowerPoint/Impress: slide, layout, temi, oggetti, note, presentazione, esportazione.
- File system: file, cartella, directory, path, estensione, copia, spostamento, rinomina, eliminazione, cestino, ricerca.
- Sistema operativo: desktop, finestre, processi, utenti, impostazioni, periferiche, aggiornamenti.
- Hardware: CPU, RAM, ROM, memoria di massa, SSD/HDD, USB, periferiche input/output, unita di misura.

## Note concorsuali

Nei quiz la risposta corretta spesso dipende da una parola tecnica: "cartella" non e' un file, "estensione" non e' il nome completo, RAM e ROM non hanno la stessa funzione, un foglio elettronico non coincide con un database relazionale. Il manuale deve allenare queste distinzioni operative.

## Note di review

- Le guide LibreOffice sono una base neutra e aperta; dove i bandi citano Microsoft Office, convertire i concetti su Word/Excel/PowerPoint senza legare il manuale a schermate di una singola versione.
- Per hardware e architettura PC serve una sintesi didattica interna: le fonti scaricate coprono gli obiettivi, ma i quiz richiedono definizioni rapide e confronti.

# Style Register Library

Reference catalog of 2D mograph illustration registers. Use when picking your 5 styles in §4 Stage 1. Each register is a real-world aesthetic with established studio touchstones — pick combinations that span 5 different creative roles, not 5 flavors of one look.

The 5 creative ROLES to fill (from SKILL.md):
1. **Safe / brand-friendly** — clean, broadly readable, low risk
2. **Bold / editorial** — visually arresting, takes a stance
3. **Playful / charming** — warmth, character, humor
4. **Minimalist / restrained** — negative space, design discipline
5. **Experimental / texture-heavy** — medium-forward, distinctive

---

## Register Catalog

### Flat Vector / Geometric
- **Look:** Clean shapes, hard edges, solid fills, geometric construction. No texture, no grain.
- **Touchstones:** Buck, Ueno, Cub Studio, Animography
- **Best for:** Tech/SaaS brand films, infographics, explainer videos
- **Roles it fills:** Safe, Minimalist
- **Prompt tags:** "flat vector illustration", "geometric construction", "solid color fills", "hard-edged shapes", "no shading or posterized two-tone shading"

### Editorial Gouache / Painterly
- **Look:** Visible brushwork, soft edges, layered color, paper-like texture
- **Touchstones:** Sehsucht, Hornet, illustrators like Olimpia Zagnoli
- **Best for:** Editorial brand films, lifestyle, premium positioning
- **Roles it fills:** Bold, Experimental
- **Prompt tags:** "soft gouache illustration", "visible brushwork", "painterly texture", "layered color washes", "paper grain"

### Hand-Drawn Cel / Wobbly Line
- **Look:** Frame-by-frame redraws, line boil, imperfect contours, traditional 2D animation feel
- **Touchstones:** Cartoon Saloon, Studio AKA, Tomer Hanuka
- **Best for:** Storytelling, character-driven content, indie/cultural brands
- **Roles it fills:** Playful, Experimental
- **Prompt tags:** "hand-drawn cel animation", "frame-by-frame style", "line boil", "wobbly contours", "12fps animation feel"

### Mixed-Media Collage / Paper Cut-Out
- **Look:** Layered paper cutouts, scanned textures, ransom-note typography, tactile assembly
- **Touchstones:** Animade, Oddfellows, Jamie Hewlett
- **Best for:** Lo-fi, indie, music, youth-culture, music videos
- **Roles it fills:** Playful, Experimental
- **Prompt tags:** "mixed-media collage", "paper cut-out illustration", "layered scanned textures", "tactile assemblage", "tape and tear edges"

### Riso / Screen-Print / Halftone
- **Look:** Limited 2–4 color palette, halftone dot patterns, slight misregistration, print-feel
- **Touchstones:** Polyester Studio, Seb Lester (print), Ill-Studio
- **Best for:** Editorial, music, fashion-adjacent, indie cultural
- **Roles it fills:** Bold, Experimental
- **Prompt tags:** "riso print illustration", "halftone dot pattern", "limited 3-color palette", "print misregistration", "screen-print feel"

### Bold Graphic Poster / Swiss-Design
- **Look:** Strong typography, asymmetric grid, primary palette, blocky shapes, design-forward
- **Touchstones:** Build (London), Sagmeister, Müller-Brockmann lineage
- **Best for:** Cultural institutions, editorial, design-aware brands
- **Roles it fills:** Bold, Minimalist
- **Prompt tags:** "swiss-design influenced graphic poster", "asymmetric grid composition", "primary color palette", "strong typographic hierarchy", "blocky geometric shapes"

### Kinetic Typography-Led
- **Look:** Type as the primary visual element, expressive layout, motion-driven word play
- **Touchstones:** Gentleman Scholar, Animography, Ariel Costa
- **Best for:** Quote-driven content, brand voice pieces, lyric videos
- **Roles it fills:** Bold, Minimalist
- **Prompt tags:** "kinetic typography composition", "type-as-image", "expressive typographic layout", "minimal supporting graphics"

### Hybrid 2D-on-3D / Volumetric Flat
- **Look:** Flat 2D illustration applied to 3D forms; reads as 2D but has subtle volume
- **Touchstones:** Giant Ant, Cub Studio (some pieces), Buck (some pieces)
- **Best for:** Premium tech, editorial product films, abstract data viz
- **Roles it fills:** Safe, Bold
- **Prompt tags:** "flat 2D textures on subtly-volumetric forms", "graphic illustration with hint of dimension", "no realistic shading, only graphic gradients"

### Glitch / CRT / Digital-Degraded
- **Look:** Lo-fi digital artifacts, scanlines, chromatic aberration, distortion, VHS grain
- **Touchstones:** ManvsMachine (some pieces), Vasava, glitch-art lineage
- **Best for:** Tech-forward, Web3, music, youth/internet-native
- **Roles it fills:** Bold, Experimental
- **Prompt tags:** "glitch art aesthetic", "CRT scanlines", "chromatic aberration", "VHS grain texture", "digital degradation artifacts"

### Children's Book Illustration / Soft
- **Look:** Warm palette, rounded shapes, hand-drawn warmth, friendly characters
- **Touchstones:** Oddfellows, Mac Premo, Jon Klassen lineage
- **Best for:** Healthcare, fintech (trust), education, family-facing brands
- **Roles it fills:** Safe, Playful
- **Prompt tags:** "children's book illustration style", "warm rounded shapes", "soft hand-drawn warmth", "friendly character design", "muted earthy palette"

### Comic / Manga / Line-and-Tone
- **Look:** Strong outlines, screen-tone shading, panel-aware composition, dynamic poses
- **Touchstones:** Studio Trigger lineage, Tomer Hanuka, Ben Wheele
- **Best for:** Action, narrative, youth, gaming-adjacent
- **Roles it fills:** Bold, Playful
- **Prompt tags:** "comic illustration style", "strong line work", "screen-tone shading", "panel-aware composition", "dynamic poses"

### Geometric Abstract / Bauhaus-Influenced
- **Look:** Pure shapes, primary palette, modernist composition, no representational content
- **Touchstones:** Bauhaus lineage, Field.io, Studio Dumbar
- **Best for:** Cultural, abstract concept films, art-forward brands
- **Roles it fills:** Minimalist, Experimental
- **Prompt tags:** "Bauhaus-influenced geometric abstraction", "pure shape composition", "primary color palette", "modernist design discipline"

## Selection Strategy

When picking 5 for a brief:

1. **Read the vibe anchor first.** "Energetic and bold" suggests Riso/Halftone, Glitch, or Comic for the bold slot. "Calm and trustworthy" suggests Flat Vector, Soft Children's, or Volumetric Flat.

2. **Read the platform.** Reels/TikTok favor higher-contrast registers (Halftone, Bold Graphic, Glitch) that read on small screens. Web hero pieces can go more refined (Gouache, Bauhaus-Abstract).

3. **Read the audience.** B2B tech → Flat Vector + Volumetric Flat are safe bets. Youth/cultural → Collage + Glitch + Halftone. Editorial/premium → Gouache + Swiss + Bauhaus-Abstract.

4. **Spread across roles intentionally.** Don't pick three flat-vector variations. The user picks visually, so visual difference is the whole point.

5. **Always include one safer pick.** Even on bold briefs — gives the user a fallback if the experimental options miss.

---

## What NOT to Do

- ✗ Pick five styles that are five tweaks of the same look
- ✗ Pick five "experimental" styles when the brief is brand-conservative
- ✗ Pick five "safe" styles when the brief explicitly wants bold
- ✗ Use "Apple-style minimal" or any non-specific aesthetic shorthand
- ✗ Tell the user which 10 you considered — just commit to the 5 you generate
